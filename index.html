About

The IPA Utility is a combination notation keyboard and reader to allow laypeople like me to use the International Phonetic Alphabet (IPA) for whatever reason. I wanted to spell out the pronouciation of my name without having to search through an IPA symbol table.

Using Wikipedia for reference, IPA symbols have been arranged by letter relevancy. For example, aɪ is a sound that's associated with the letter i, so it can be found by hovering over the i key. dʒ is a sound that can come from the letter g or j, so you can click it from either key. These associations are not part of the IPA. They were determined by me personally in an effort to devise and implement a solution that's layperson-friendly. Additionally, word examples for symbol pronouciation are provided next to each symbol. You can also interact with the on-screen keyboard using tab and space/enter on your physical keyboard.
