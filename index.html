IPA Keyboard

the International Phonetic Alphabet for laypeople

About

The IPA Keyboard allows laypeople like me to use the IPA without having to search through an IPA symbol table.

Using Wikipedia, IPA symbols have been arranged by letter relevancy. For example, aɪ is a sound that's associated with the letter i, so it can be found by hovering over the i key. dʒ is a sound that can come from the letter g or j, so you can click it from either key. These associations aren’t official or anything, I just created them using my best judgment in an effort to devise and implement this layperson-friendly interface. Additionally, word examples for symbol pronunciation are provided next to each symbol. You can also interact with the on-screen keyboard using tab and space/enter on your physical keyboard.
